Contributing image sets

If you would like to contribute a dataset, please fill out this form: https://goo.gl/forms/qBdgPcZiEodKDI7E2. Once you have submitted the form, please email imagingadmin@broadinstitute.org with the following:
  • a zip file of your images
  • 1-5 example images
  • any metadata or ground truth data you may have (.csv files).
Please email imagingadmin@broadinstitute.org if you have any questions about contributing datasets. {% endblock %}
