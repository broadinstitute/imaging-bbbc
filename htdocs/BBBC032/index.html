Mouse embryo blastocyst cells

Accession number BBBC032 · Version 1

Example images

Description of the biological application

Mouse embryo blastocyst cells marked with single molecule FISH for BMP4 and counter-stained with WGA)

Images

Images acquired by using PerkinElmer Ultraview VoX spinning disk microscope combined with a Leica SP8 with a 63x immersion objective (Distance between Z-slices = 0.5 um)

  • 647 channel (BMP4blastocystC0): BMP4 transcripts
  • 568 channel (BMP4blastocystC1): GAPD transcripts
  • 488 channel (BMP4blastocystC2): WGA
  • 405 channel (BMP4blastocystC3): Hoechst
  • Width = 103.424 microns (1024)
  • Height = 135.744 microns (1344)
  • Depth = 86 microns (172)

    • BBBC031_v1_dataset.zip (56.3MB)
